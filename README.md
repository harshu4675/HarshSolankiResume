# Harsh Solanki — Portfolio Website

Premium portfolio website built as a modern software product showcase.

![Portfolio Preview](./client/public/assets/og-image.jpg)

## 🚀 Tech Stack

### Frontend

- **React 18** + **Vite** — Fast development & optimized builds
- **Tailwind CSS** — Utility-first styling with custom design system
- **Framer Motion** — Smooth, premium animations
- **React Hook Form** — Performant form handling
- **React Hot Toast** — Notification system

### Backend

- **Node.js** + **Express.js** — REST API server
- **MongoDB** + **Mongoose** — Database & ODM
- **JWT** — Authentication
- **Nodemailer** — Email notifications
- **Helmet + Rate Limiting** — Security
